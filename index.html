Jersey Modules

jersey-container

Jersey Container API. Discover the implementation to use from the classpath.

Container container = Container.create(GuestfulResourceConfig)
    .setPort(Env.PORT)
    .setMaxWorkers(Env.WORKERS)
    .setContextPath('/api')
